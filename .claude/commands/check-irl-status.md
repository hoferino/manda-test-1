Execute the Check IRL Status workflow.

1. Load the workflow configuration: `workflows/check-irl-status/workflow.yaml`
2. Load the workflow engine: `workflows/workflow-engine.xml`
3. Load the workflow instructions: `workflows/check-irl-status/instructions.md`

This workflow will:
- Load the IRL checklist tracking file
- Calculate completion percentages by category
- Identify high-priority gaps
- Show document collection progress
- Generate status dashboard

Quick way to monitor your data room completeness and identify what's still needed.
